# New Startup Portal App
A PERN Stack Application for New Startup Platform

## Getting Started
### Requirments
* Docker
* Node.js
* npm

### Excuting App (Optional: Using Docker)
```bash
# Build images
docker-compose build

# Run services
docker-compose up

# Stop sevices
docker-compose down
```
